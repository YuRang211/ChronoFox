# Fox Calendar

Python `PySide6`로 만든 바탕화면용 투명 달력 + 독립 스티커 메모입니다.
달력은 일요일부터 토요일까지의 순서로 표시됩니다.

## 실행 방법

1. VS Code에서 이 폴더를 엽니다.
2. 터미널에서 아래 명령을 실행합니다.

```powershell
python -m pip install PySide6
python .\desktop_note_calendar.py
```

## 사용 방식

- 날짜를 누르면 해당 날짜의 일정을 적는 창이 열립니다.
- 달력 칸에는 저장한 일정의 앞부분이 표시됩니다.
- 긴 일정은 칸 밖으로 넘치지 않고 `...`로 줄여 표시됩니다.
- `새 메모` 버튼을 누르면 날짜와 상관없는 스티커 메모가 따로 열립니다.
- 메모 내용과 열린 메모 창 위치는 자동 저장됩니다.
- 프로그램을 다시 켜면 마지막에 열려 있던 메모 창이 다시 뜹니다.
- 기본 저장 위치는 `문서\DesktopNotes`입니다.
- 이동 아이콘 버튼을 좌클릭한 채 드래그하면 달력 위치를 옮길 수 있습니다.
- 달력 오른쪽 아래 `◢`를 드래그하면 달력 크기를 바꿀 수 있습니다.
- 노트 아이콘 버튼을 누르면 새 스티커 메모가 열립니다.
- 설정 아이콘 버튼을 누르면 설정창이 열립니다.
- 설정창에서 다크모드 켜기, 끄기, Windows 설정 따르기를 선택할 수 있습니다.
- 주요 고정 공휴일은 자동으로 달력에 표시됩니다.
- 메모 창의 `이동` 버튼을 좌클릭한 채 드래그하면 메모 위치를 옮길 수 있습니다.
- 메모 창 오른쪽 아래 모서리를 드래그하면 크기를 바꿀 수 있습니다.
- 우클릭 메뉴에서 Windows 시작 시 자동 실행을 켜거나 끌 수 있습니다.
- `Esc` 키 또는 `x` 버튼으로 닫을 수 있습니다.

## 노트 앱과 연동하기

Obsidian처럼 Markdown 폴더를 그대로 여는 노트 앱이라면, `Notes Folder`를 해당 vault 또는 노트 폴더로 지정하면 됩니다.

Notion, OneNote, Google Keep처럼 자체 API나 클라우드 동기화를 쓰는 앱은 다음 단계에서 별도 연동 모듈을 붙이는 방식이 좋습니다.
