# ChronoFox

크로노폭스 v0.7은 Python `PySide6`로 만든 Windows 바탕화면용 달력 + 스티커 메모 + 시간 관리 앱입니다.

언어는 Windows 시스템 언어를 기준으로 처음 설정되며, 앱 안에서는 `설정 > 테마 > 언어`에서 한국어/영어를 바꿀 수 있습니다.

## 실행 방법

1. VS Code에서 이 폴더를 엽니다.
2. 터미널에서 아래 명령을 실행합니다.

```powershell
python -m pip install PySide6
python .\desktop_note_calendar.py
```
